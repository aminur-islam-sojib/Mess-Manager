 ## Mess Manager
  Mess Manager is a role-based web application built with Next.js and MongoDB that helps manage shared mess/hostel expenses efficiently.
The system supports two roles — Manager and User — where managers can create a mess and invite members via secure email invitations, while users can join a mess, participate in meal tracking, and request approval for shared expenses.

The project is designed with real-life mess management in mind and focuses on scalability, clean architecture, and production-ready practices. Future features include meal cost calculation, shopping expense approval, and monthly expense reports.

Tech Stack: Next.js (App Router), TypeScript, Tailwind CSS, MongoDB, Mongoose, Role-Based Authentication.
